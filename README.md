# DSDESPACHANTEPAGINA1

Site profissional para DS Despachante Guarujá, focado em regularização veicular e cursos EAD.

## Como usar as imagens
Para que as fotos apareçam no site, você deve:
1. Criar a pasta `public/images/` na raiz do projeto.
2. Adicionar os seguintes arquivos:
   - `logo.png` (Logotipo)
   - `hero.jpg` (Imagem de fundo do topo)
   - `about.jpg` (Foto da seção Sobre)

## Comandos Git para este repositório
```bash
git init
git add .
git commit -m "Site completo com navegação acessível e logotipo empilhado"
git branch -M main
git remote add origin https://github.com/Cabanas01/DSDESPACHANTEPAGINA1.git
git push -u origin main
```
